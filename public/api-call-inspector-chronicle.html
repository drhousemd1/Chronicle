API Call Inspector

High-fidelity reference for every backend call in the Chronicle application. Use the Edit button in the header bar to add or modify entries.

GET /rest/v1/stories

Fetch all stories for the authenticated user. Used by the Story Hub on initial load.

POST /rest/v1/stories

Create a new story/scenario. Called from the "New Story" button in the Story Hub.

GET /rest/v1/characters

Fetch characters for a given scenario. Used by the Characters tab and Chat sidebar.

POST /rest/v1/conversations

Create a new conversation/session for a scenario. Called when starting a new chat.

GET /rest/v1/messages

Fetch message history for a conversation. Loaded when opening an existing chat session.

POST /functions/v1/chat

Send a user message and receive an AI-generated response. Core chat endpoint.

POST /functions/v1/generate-cover-image

Generate a cover image for a story using AI image generation.

POST /functions/v1/generate-scene-image

Generate a scene illustration during chat based on narrative context.

GET /rest/v1/art_styles

Fetch available art styles for image generation. Used by the Art Style picker.

GET /rest/v1/app_settings

Fetch application-wide settings (shared API keys, admin tool metadata, etc.).

PUT /rest/v1/app_settings

Update an application setting. Used by Admin tools to persist configuration.

GET /rest/v1/profiles

Fetch user profile data (display name, avatar, preferences). Used by Account page.

POST /functions/v1/extract-memory-events

Extract notable events from chat messages for the memory system.

DELETE /rest/v1/stories?id=eq.{id}

Delete a story and all associated data (characters, conversations, messages).

More endpoints will be documented as the application grows. Use the Edit mode to add new entries.
